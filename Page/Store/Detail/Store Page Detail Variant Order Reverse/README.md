# Store Product Detail Variant Order Reverse

### [License][1]

### Synopsis

Reverse the order of product detail variants.

### Version

  * 1.0.1

#### SS Versions

  * 7.1

  * 7.0 ([Brine template family][2])

#### v7.1 Products V2 Compatible

  * Yes
  
#### v7.1 Fluid Engine Compatible

  * Not Applicable

---

## Install

* Options

  * Page Specific
  
    * Use this option if you have only a single store page you want to have this
      effect.
      
    * Add code from file **[store page detail variant order reverse.html][3]**
      to Store Page Settings > Advanced > Page Header Code Injection for the
      page.
      
    * Refer to [per-page code injection][4] for details.
      
  * Site-wide
  
    * Use this option if you have multiple store pages where you want to have
      this effect.
      
    * v7.1
    
      * Add code from file **[store page detail variant order
        reverse.less][5]** to Website > Pages > Custom Code > Custom CSS.
        
    * v7.0
    
      * Add code from file **[store page detail variant order
        reverse.less][5]** to Website > Pages > Website Tools > Custom CSS.
        
    * Refer to [Using the CSS Editor][6] for details.

# Store Page Detail Slideshow Item Badge Add

### [License][1]

### Synopsis

Add badges to product detail slideshow badges.

### Version

  * 0.3.0

#### SS Versions

  * 7.1
  
  * 7.0 ([Brine template family][2])

#### v7.1 Products V2 Compatible

  * Yes
  
#### 7.1 Fluid Engine Compatible

  * Not Applicable

#### Dependencies

  * [Squarespace plan][3] that supports [JavaScript][4].

---

## Quick Install

* Options

  * Page Specific
  
    * Use this option if you have only a single store page you want to have this
      effect.
      
    * Add code from file **[store page detail slideshow item badge
      add.html][14]** to Store Page Settings > Advanced >
      Page Header Code Injection for the page.
      
    * Refer to [per-page code injection][5] for details.
      
  * Site-wide
  
    * Use this option if you have multiple store pages where you want to have
      this effect.
      
    * v7.1
    
      * Add code from file **[store page detail slideshow item badge
        add.html][14]** to Website > Pages > Custom Code > Code Injection >
        FOOTER.
        
    * v7.0
    
      * Add code from file **[store page detail slideshow item badge
        add.html][14]** to Website > Pages > Website Tools > Custom Code >
        Code Injection > FOOTER.
        
    * Refer to [Add code to code injection][6].

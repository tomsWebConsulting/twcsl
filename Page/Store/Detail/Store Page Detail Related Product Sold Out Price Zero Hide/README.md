# Store Page Detail Related Product Sold Out Price Zero Hide

### [License][1]

### Synopsis

Hide related product zero price when sold out.

### Version

  * 0.2.0

#### SS Versions

  * 7.1

  * 7.0 ([Brine template family][2], [Five][3], [Montauk template family][4])

#### v7.1 Products V2 Compatible

  * Yes
  
#### v7.1 Fluid Engine Compatible

  * Not Applicable

#### Dependencies

  * [Squarespace plan][5] that supports [JavaScript][6].

---

## Install

* Install [Store Page Price Change][7]. After you install it be sure to
  continue on with the rest of the steps.
  
* Options

  * Page Specific
  
    * Use this option if you have only a single store page you want to have this
      effect.
      
    * Add code from file **[store page detail related product sold out price
      zero hide.html][8]** to Store Page Settings > Advanced > Page Header Code
      Injection for the page.
      
    * Refer to [per-page code injection][9] for details.
      
  * Site-wide
  
    * Use this option if you have multiple store pages where you want to have
      this effect.
      
    * v7.1
    
      * Add code from file **[store page detail related product sold out price
        zero hide.html][8]** to Website > Pages > Custom Code > Code Injection >
        FOOTER.
        
    * v7.0
    
      * Add code from file **[store page detail related product sold out price
        zero hide.html][8]** to Website > Pages > Website Tools > Custom Code >
        Code Injection > FOOTER.
        
    * Refer to [Add code to code injection][10].

# Page Settings SEO to Default

### [License][1]

### Synopsis

Reset Page SEO settings to default.

### Version

  * 0.1.0

#### SS Versions

  * 7.1
  
  * 7.0

---

## Quick Install

Add code from file **[page settings seo to default.bookmarklet][2]** as the URL
for a bookmark in your browser named **TWC Page Page Settings SEO to Default**.

## How to Use

* Select a Page in the Pages Panel.

* Use the bookmarklet.

## What is a bookmarklet?

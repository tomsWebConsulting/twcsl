# Blog Page List Url Slug Last Dash Text Remove

### [License][1]

### Synopsis

Remove the last dash text from end of post URL Slugs.

### Version

  * 0.1.0

#### SS Versions

  * 7.1
  
  * 7.0

---

## Quick Install

Add code from file **[blog page list url slug last dash text
remove.bookmarklet][2]** as the URL for a bookmark in your browser named **TWC
Blog Page List Url Slug Last Dash Text Remove**.

## How to Use

Squarespace sometimes adds unique suffixes to the end of Blog Post URL Slugs.
For example **blog-post-one-t5my5-k4xmd-67jzh-srp3y**. This can cause issues
either with the clarity of or preserving URL Slugs. 

This bookmarklet works on a single Blog list page basis. In other words it will
not fix all your bad URLs in one go. This is for caution sake. If something does
go wrong with the code then it should only damage a maximum of 20 posts.

* Go to a Blog Page list.

* Use the bookmarklet.

* The bookmarklet will run and if succesfull the Page will reload.

## Notes

This bookmarklet has the **potential to cause loss of data**. Use at your own
risk!

I suggest you proceed with extreme caution and work on test posts at first to
get comfortable using this bookmarklet.

I can't tell you how to best use this bookmarklet in a process of fixing issues.
I can say that this is a blunt instrument. The bookmarklet is not smart. It
does not know which Blog Posts have been processed. It simply does what it says
it will do. Remove the last dash and text at the end of a URL Slug.

# Shipping Options Remove All

### [License][1]

### Synopsis

Remove all Shipping Options from Settings > Selling > Shipping > Shipping
Options.

### Version

  * 0.1.0

#### SS Versions

  * 7.1
  
  * 7.0

---

## Quick Install

Add code from file **[shipping options remove all.bookmarklet][2]** as the URL
for a bookmark in your browser named **TWC Shipping Options Remove All**.

## How to Use

* Go to Settings > Selling > Shipping.

* Use the bookmarklet.

## What is a bookmarklet?

Wikipedia has a nice entry on [bookmarklet][3].

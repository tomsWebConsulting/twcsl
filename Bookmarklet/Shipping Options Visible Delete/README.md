# Shipping Options Visible Delete

### [License][1]

### Synopsis

Delete visible Shipping Options from Settings > Selling > Shipping > Shipping
Options.

### Version

  * 0.1.1

#### SS Versions

  * 7.1
  
  * 7.0

---

## Quick Install

Add code from file **[shipping options visible delete.bookmarklet][2]** as the
URL for a bookmark in your browser named **TWC Shipping Options Visible
Delete**.

## How to Use

* Go to Settings > Selling > Shipping.

* Use the bookmarklet.

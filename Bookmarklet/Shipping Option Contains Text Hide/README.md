# Shipping Option Contains Text Hide

### [License][1]

### Synopsis

Hide Shipping Option containing text.

### Version

  * 0.1.0

#### SS Versions

  * 7.1
  
  * 7.0

---

## Quick Install

Add code from file **[shipping option contains text hide.bookmarklet][2]** as
the URL for a bookmark in your browser named **TWC Shipping Option Contains Text
Hide**.

## How to Use

* Go to Settings > Selling > Shipping.

* Use the bookmarklet.

## What is a bookmarklet?

Wikipedia has a nice entry on [bookmarklet][3].

## Note

To make your hidden shipping options visible again, reload the page.

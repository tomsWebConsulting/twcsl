# Modal Lightbox Fix

### [License][1]

### Synopsis

Fix Squarespace's incorrect CSS for lightboxes

### Version

  * 1.0.0

#### SS Versions

  * 7.1
  
  * 7.0

---

## Install Options

* CDN Hosted

  Use this option for the quickest way to install this effect (files hosted
  externally on the [jsDelivr][2], a [CDN][3])
  
  * Options
  
    * Page Specific
    
      Use this option if you want to have this effect on only one Page.
      
      * Add the following code to Page Settings > Advanced >
        Page Header Code Injection for the Page.
        
        ```html
        <!-- begin TWC Modal Lightbox Fix -->
        
          <!-- License < https://github.com/tomsWebConsulting/twcsl/blob/main/LICENSE.txt#L1 > -->
          
          <link href="https://cdn.jsdelivr.net/gh/tomsWebConsulting/twcsl@96e8dc387753ee13adc372ebc34d2adeed55f29f/Element/Modal%20Lightbox%20Fix/modal%20lightbox%20fix.min.css" rel="stylesheet" type="text/css">
          
          <!-- end TWC Modal Lightbox Fix -->
          
        ```
        
      * Refer to [per-page code injection][4] for details.
      
    * Site-wide
    
      Use this option if you want to have this effect on all Pages.
      
      * Add the following code to Website > Pages > Custom Code >
        Code Injection > HEADER.
        
        ```html
        <!-- begin TWC Modal Lightbox Fix -->
        
          <!-- License < https://github.com/tomsWebConsulting/twcsl/blob/main/LICENSE.txt#L1 > -->
          
          <link href="https://cdn.jsdelivr.net/gh/tomsWebConsulting/twcsl@96e8dc387753ee13adc372ebc34d2adeed55f29f/Element/Modal%20Lightbox%20Fix/modal%20lightbox%20fix.min.css" rel="stylesheet" type="text/css">
          
          <!-- end TWC Modal Lightbox Fix -->
          
        ```
            
        * Refer to [Add code to code injection][5] for details.
        
* On-site

  Use this option to install the full code of this effect (files hosted on your
  site).
  
  * Page Specific
  
    Use this option if you want to have this effect on only one Page.
    
    * Add code from file **[modal lightbox fix.html][6]** to Page Settings >
      Advanced > Page Header Code Injection for the Page.
      
    * Refer to [per-page code injection][4] for details.
    
  * Site-wide
  
    Use this option if you want to have this effect on all Pages.
    
      * Add code from file **[modal lightbox fix.html][6]** to Website >
        Pages > Custom Code > Code Injection > FOOTER.
        
      * Refer to [Add code to code injection][5] for details.

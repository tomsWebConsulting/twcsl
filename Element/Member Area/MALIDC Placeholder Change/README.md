# MALIDC Placeholder Change

### [License][1]

### Synopsis

Change placeholder text of member area member sites join dialog.

### Version

  * 0.1.0

#### SS Versions

  * 7.1
  
  * 7.0

#### Dependencies

  * [Squarespace plan][2] that supports [JavaScript][3].
  
  * [Member Area Log In Dialog Change][4]

#### Fluid Engine Compatible

  * Not Applicable

---

## Install

* Install [Member Area Log In Dialog Change][5]. After you install the code be
  sure to continue on with the rest of the steps.
  
* Add the code from file **[malidc placeholder change.html][6]** to Website >
  Pages > Website Tools > Code Injection > FOOTER. Refer to [Per-page code
  injection][7] for details.

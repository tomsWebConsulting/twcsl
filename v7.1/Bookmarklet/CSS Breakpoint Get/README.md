# CSS Breakpoint Get

### [License][1]

### Synopsis

Get the CSS breakpoint value from the site.

### Version

  * 0.1.0

#### SS Version

  * 7.1

---

## Quick Install

Add code from file **[css breakpoint get.bookmarklet][2]** as the URL for a
bookmark in your browser named **TWC CSS Breakpoint Get**.

## How to Use

* In the Squarepace site editor [click on the Site Styles button][3].

* Go to Miscellaneous > Spacing.

* Change Page With(Max) or Site Margin.
      
* Undo that change but do not SAVE or EXIT.
      
* Use this bookmarklet.

* Note the breakpoint value.

* Exit Site Styles without saving changes.

# CSS Breakpoint Get

### [License][1]

### Synopsis

Get the CSS breakpoint value from the site.

### Version

  * 0.2.0

#### SS Version

  * 7.1

---

## Quick Install

Add code from file **[css breakpoint get.bookmarklet][2]** as the URL for a
bookmark in your browser named **TWC CSS Breakpoint Get**.

## How to Use

While in the Squarepace site editor use the bookmarklet.

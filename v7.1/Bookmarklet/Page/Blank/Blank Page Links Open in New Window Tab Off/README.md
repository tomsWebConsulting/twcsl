# Blank Page Links Open in New Window Tab Off

### [License][1]

### Synopsis

Turn off links Open in New Tab setting.

### Version

  * 0.1.0

#### SS Version

  * 7.1

---

## Quick Install

Add code from file **[blank page links open in new window tab
off.bookmarklet][2]** as the URL for a bookmark in your browser named **TWC
Blank Page Links Open in New Window Tab Off**.

## How to Use

* Select a Blank Page in the Pages Panel.

* Use the bookmarklet.

* Wait for the page to reload. The larger the page the more time it takes for
  the reload to occur.

## What is a bookmarklet?

Wikipedia has a nice entry on [bookmarklet][5].

## Note

This bookmarklet does not work with Gallery Sections.

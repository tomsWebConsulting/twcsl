# Store Page Categories Add

### [License][1]

### Synopsis

Add a comma separated string as categories in the Store page Categories panel.

### Version

  * 0.1.0

#### SS Version

  * 7.1

#### Products V2 Compatible

  * Not Applicable
  
#### Fluid Engine Compatible

  * Not Applicable

---

## Quick Install

Add code from file **[store page categories add.bookmarklet][2]** as the URL for
a bookmark in your browser named **TWC Store Page Categories Add**.

## How to Use

* Prepare a comma separated string of categories and get it into your clipboard.

* Go to Store Settings > Categories.

* Paste the clipboard contents into the **Add or Search for a Category** field.

* Use the bookmarklet.

* Wait for the bookmarklet to enter your categories.

## Note

You can include the comma character in your categories by using typing **\\,**
(backslash comma).

# Pages Panel Collapse All

### [License][1]

### Synopsis

Collapse all pages in the Pages panel.

### Version

  * 0.1.1

#### SS Version

  * 7.1

---

## Quick Install

* Add code from file **[pages panel collapse all.bookmarklet][2]** as the URL
  for a bookmark in your browser named **TWC Pages Panel Collapse All**.

## How to Use

* Navigate to the Pages panel of your site.

* Use the bookmarklet.

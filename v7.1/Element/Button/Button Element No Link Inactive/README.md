# Button Element No Link Inactive

### [License][1]

### Synopsis

Make Button elements with no link inactive.

### Version

  * 0.1.0

#### SS Version

  * 7.1

---

## Install Options

* CDN Hosted

  Use this option for the quickest way to install this effect (files hosted
  externally on the [jsDelivr][2], a [CDN][3])
  
  * Options
  
    * Page Specific
    
      Use this option if you want to have this effect on only one Page.
      
      * Add the following code to Page Settings > Advanced >
        Page Header Code Injection for the Page.
        
        ```html
        <!-- begin TWC Button Element No Link Inactive -->
        
          <!-- License < https://github.com/tomsWebConsulting/twcsl/blob/main/LICENSE.txt#L1 > -->
          
          <link href="https://cdn.jsdelivr.net/gh/tomsWebConsulting/twcsl@d1a2a637af737cfd932c328cee2d7952e3642538/Element/Button/Button%20Element%20No%20Link%20Inactive/button%20element%20no%20link%20inactive.min.css" rel="stylesheet" type="text/css">
          
          <!-- end TWC Button Element No Link Inactive -->
          
        ```
        
      * Refer to [per-page code injection][4] for details.
      
    * Site-wide
    
      Use this option if you want to have this effect on all Pages.
      
      * Add the following code to Website > Pages > Custom Code >
        Code Injection > HEADER.
        
        ```html
        <!-- begin TWC Button Element No Link Inactive -->
        
          <!-- License < https://github.com/tomsWebConsulting/twcsl/blob/main/LICENSE.txt#L1 > -->
          
          <link href="https://cdn.jsdelivr.net/gh/tomsWebConsulting/twcsl@d1a2a637af737cfd932c328cee2d7952e3642538/Element/Button/Button%20Element%20No%20Link%20Inactive/button%20element%20no%20link%20inactive.min.css" rel="stylesheet" type="text/css">
          
          <!-- end TWC Button Element No Link Inactive -->
          
        ```
            
        * Refer to [Add code to code injection][5] for details.
        
* On-site

  Use this option to install the full code of this effect (files hosted on your
  site).
  
  * Page Specific
  
    Use this option if you want to have this effect on only one Page.
    
    * Add code from file **[button element no link inactive.html][6]** to
      Page Settings > Advanced > Page Header Code Injection for the Page.
      
    * Refer to [per-page code injection][4] for details.
    
  * Site-wide
  
    Use this option if you want to have this effect on all Pages.
    
    * Add code from file **[button element no link inactive.css][7]** to
      Website > Pages > Custom Code > Custom CSS.
      
    * Refer to [Using the CSS Editor][8] for details.

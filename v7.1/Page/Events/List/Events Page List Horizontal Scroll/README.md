# Events Page List Horizontal Scroll

### [License][1]

### Synopsis

Reformat Events page list for horizontal scroll.

### Version

  * 0.1.0

#### SS Version

  * 7.1

#### Fluid Engine Compatible

  * Not Applicable

#### Dependencies

  * [Squarespace plan][2] that supports [JavaScript][3].

---

## Quick Install

* Options

  * Page Specific
  
    Use this option when you want only one Page to have this effect.
    
    * Add code from file **[events page list horizontal scroll.html][4]** to
      Page Settings > Advanced > Page Header Code Injection for the Page. Refer
      to [Add code to code injection][5] for details.
      
  * Site-wide
  
    Use this option when you want multiple Pages to have this effect.
    
    * Add code from file **[events page list horizontal scroll.less][6]** to
      Website > Pages > Custom Code > Code Injection > Custom CSS.
      
* Read the code for any instructions within.

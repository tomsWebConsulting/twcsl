# Store Page Product Dimensions

### [License][1]

### Synopsis

Override shipping dimensions.

### Version

  * 0.1.0

#### SS Version

  * 7.1

#### Products V2 Compatible

  * Not Applicable
  
#### Fluid Engine Compatible

  * Not Applicable

#### Dependencies

  * [Squarespace plan][2] that supports [JavaScript][3].

---

## Install
    
* Store Page Specific

  * Use this option if you want to have this effect on only one Store page.
    
  * Add code from file **[store page product dimensions.html][4]** to
    Store Settings > Advanced > Page Header Code Injection for the Page.
    
  * Add code from file **[store page product dimensions inventory.html][5]** to
    Store Settings > Advanced > Page Header Code Injection for the Page. Read
    the code for any instructions within.
    
  * Refer to [per-page code injection][6] for details.
  
* Site-wide

  * Use this option if you want to have this effect on all Store pages.
    
  * Add code from file **[store page product dimensions.html][4]** to Website >
    Pages > Custom Code > Code Injection > HEADER.
    
  * Add code from file **[store page product dimensions inventory.html][5]** to
    Website > Pages > Custom Code > Code Injection > HEADER. Read the code for
    any instructions within.
    
  * Refer to [Add code to code injection][7] for details.

## Notes

This code on its own has no visible effect. It is designed to provide product
dimensions to other effects.

The dimensions entered in the Squarespace editor are shipping dimensions, not
product dimensions.
